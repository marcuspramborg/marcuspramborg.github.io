# ADHD/ADD Exam Support Tools

A web-based accessibility toolkit designed to help college students with ADHD and ADD during examinations. This tool provides helpful features that support focus and organization without giving unfair academic advantages.

## Features

### üéØ Core Tools

- **Focus Timer**: Pomodoro-style timer to help manage time and maintain focus
- **Progress Tracker**: Visual progress bar to track exam completion
- **Quick Notes**: Space for jotting down thoughts, reminders, or scratch work
- **Reading Support**: Adjustable text size, line spacing, and high contrast mode
- **Text-to-Speech**: Upload text files or paste content to have it read aloud
- **Stress Management**: 4-7-8 breathing exercise for anxiety relief
- **ADHD Study Companion**: Evidence-based tips and strategies for ADHD/ADD students
- **Color Coding**: Visual organization system for highlighting and categorization

### üåü Accessibility Features

- **Dark Mode**: Reduces eye strain in low-light conditions
- **High Contrast Mode**: Improves readability for visual processing difficulties
- **Keyboard Shortcuts**: Quick access to common functions
- **Responsive Design**: Works on various screen sizes
- **Local Storage**: Saves preferences and notes locally

## How to Use

1. Open `index.html` in any modern web browser
2. Adjust settings according to your needs (text size, contrast, etc.)
3. Use the timer to manage work periods
4. Track your progress through the exam
5. Take breathing breaks when feeling overwhelmed
6. Use quick notes for temporary thoughts or calculations

## Technical Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- No internet connection required after initial load
- Works offline once files are downloaded
- Compatible with screen readers

## Installation

1. Download all files (`index.html`, `styles.css`, `script.js`)
2. Place them in the same folder
3. Open `index.html` in your web browser
4. Bookmark the page for easy access during exams
